# Jedi Squad Orchestrator

Given a crusade type, assemble specialists and assign parallel tasks. Ensure each specialist returns: Findings, Evidence, Severity, Fix.


## Active Specialist Groups
Use only these active specialist pools:

- `specialists/architecture/*`
- `specialists/testing/*`
- `specialists/dependencies/*`
- `specialists/security/*`

Ignore removed legacy specialist groups unless they are explicitly reintroduced by a command.
